🧮 linhsan-learning — AI Math Tutor cho lớp 4

Mô tả dự án

linhsan-learning là ứng dụng web học toán thông minh dành cho học sinh lớp 4 Việt Nam (9–10 tuổi), triển khai trên Streamlit Community Cloud. Người dùng chọn một chủ đề toán → hệ thống dùng 2 AI Agent để tự động sinh bài toán và giải thích lời giải theo cách một bé 5 tuổi có thể hiểu được.

Quy mô: nhỏ, 1–2 người dùng — không cần database, không cần auth.

Hướng dẫn Claude implement từng module

Quy trình chuẩn

  1. Đọc PROJECT_REQUIREMENTS.md — nắm user story và acceptance criteria của module cần làm.
  2. Đọc CLAUDE.md — xem kiến trúc tổng thể, vị trí file trong cây thư mục.
  3. Implement code tại source/backend/ — đúng module, đúng class/function.
  4. Viết unit test tại unittest/backend/ — mock LLM, không gọi API thật.
  5. Cập nhật cột Trạng thái trong bảng TODO ở trên thành DONE.

Quy tắc bắt buộc

  • API key: Chỉ đọc từ st.secrets["GROQ_API_KEY"] hoặc os.environ["GROQ_API_KEY"] — không bao giờ hard-code.
  • JSON output LLM: Luôn dùng try/except json.JSONDecodeError, có fallback message.
  • Prompt: Template đặt trong source/backend/prompts/*.j2, không nhúng thẳng vào code.
  • Test: Dùng unittest.mock.patch để mock Groq client — không tốn API credit.
